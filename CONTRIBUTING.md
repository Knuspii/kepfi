# Contributing to kepfi 🛡️

First off, thanks for taking the time to contribute! Contributions are what make the open-source community such an amazing place to learn, inspire, and create.

## 🚫 The Golden Rule: No External Dependencies
To keep `kepfi` lightweight, secure, and easy to compile on any system, this project has a **strict zero-dependency policy**. 

- **Only Go Standard Library:** All logic must be implemented using the built-in packages.
- **No `go.mod` requirements:** If a feature requires an external library to work, we will either find a way to write it ourselves or skip the feature.

## Submit a Pull Request
- Describe your changes in detail.
- Explain why the change is necessary.
- Please no unnecessary changes I want this tool to be simple!
